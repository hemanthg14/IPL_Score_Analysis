# IPL Score Analysis

## 📌 Project Overview
This project analyzes IPL match scores using Python and Pandas. It calculates team-wise total runs, average runs, and highest runs scored.

## 🎯 Objectives
- Find Total Runs scored by each Team
- Find Average Runs scored by each Team
- Find Highest Runs scored by each Team

## 🛠️ Tools & Technologies
- Python
- Pandas
- CSV Dataset

## 📊 Concepts Used
- DataFrame
- GroupBy Function
- Aggregation Functions (sum, mean, max)

## 📁 Dataset Used
ipl.csv

## ▶️ How to Run
1. Install pandas library
2. Run the Python file using:
   python analysis.py
